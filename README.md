### Installation 

Clone Repository:
```
git clone https://github.com/raulduartep/happy2.0.git
```

Run Api
```
# Go to server folder
$ cd Happy2.0/server

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn dev 
```

Run Web
```
# Go to web folder
$ cd Happy2.0/web

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
